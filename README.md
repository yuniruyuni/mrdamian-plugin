# mrdamian-plugin

mrdamian-plugin is Mr.Damian plugin abstract interfaces.
Each Mr.Damian's plugin should implement this interfaces.

- Install MrDamian.
- Install [Bun](https://bun.sh/). (All Mr.Damian's plugins should be built with Bun.)
